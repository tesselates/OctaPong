# OctaPong
Scaleable pong game

## Features

Currently only supports two players

## Dependancies
QT 5.15.0

## How to run

        git clone https://github.com/tessellates/OctaPong.git
        cd OctaPong
        mkdir build
        cd build
        cmake .. -DQt5_DIR="<...>/qtbase/lib/cmake/Qt5/"
        make
        ./pong
